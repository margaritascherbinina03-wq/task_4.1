## git log
используется для просмотра истории коммитов репозитория. Она показывает хронологический список всех коммитов, от самых последних до самых ранних, предоставляя подробную информацию о каждом из них.

git log — это ваш журнал изменений проекта, который позволяет вам заглянуть в прошлое и узнать, кто, что, когда и почему изменил.
