Git — распределённая система контроля версий, созданная Линусом Торвальдсом в 2005 году для управления разработкой ядра Linux. В отличие от централизованных систем, Git хранит полную копию репозитория на компьютере каждого разработчика, что обеспечивает автономность работы без постоянного подключения к серверу. 

## Ключевые концепции:

*Репозиторий* — хранилище кода проекта со всей его историей изменений.

*Коммит* — снимок состояния проекта в определённый момент времени.

*Ветка* — независимая линия разработки, позволяющая изолировать изменения.
